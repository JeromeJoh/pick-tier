## Best Practices

### For Individual Use
1. **Prepare Elements**: Upload all images before starting
2. **Configure Tiers**: Set up your tier system first
3. **Use Keyboard**: Keyboard shortcuts are faster than clicking
4. **Take Breaks**: Exit and resume as needed

### For Presentations
1. **Test First**: Run through the presentation beforehand
2. **Use Auto-Advance**: Let the system handle timing
3. **Explain Controls**: Show audience the tier system
4. **Engage Audience**: Ask for input on rankings

### For Rapid Ranking
1. **Use Number Keys**: Fastest way to rank elements
2. **Skip Uncertain Items**: Come back to them later
3. **Enable Auto-Advance**: Maintain momentum
4. **Focus on First Impressions**: Trust your initial reaction

## Tips and Tricks

- **Mobile Friendly**: Works well on tablets and phones
- **Resume Anytime**: Exit and restart to continue where you left off
- **Batch Processing**: Perfect for large collections of elements
- **Presentation Ready**: Great for showing rankings to others
- **Distraction-Free**: Full-screen mode eliminates UI clutter

## Troubleshooting

### No Elements Available
- Make sure you have uploaded images
- Check that elements are in the pool (not already ranked)
- Try uploading new images

### Keyboard Shortcuts Not Working
- Make sure the present mode window has focus
- Click inside the presentation area
- Try using mouse controls instead

### Performance Issues
- Close other browser tabs
- Use smaller image files
- Restart the browser if needed

## Integration with Main App

Present Mode is fully integrated with the main application:
- **Rankings are permanent**: Elements ranked in present mode appear in the main tier view
- **Real-time updates**: Changes are reflected immediately
- **Consistent data**: No data loss when switching between modes
- **Seamless workflow**: Use alongside drag-and-drop ranking

Present Mode provides a focused, efficient way to rank large collections of elements while maintaining the flexibility and power of the main Pick Tier application.
